About us and Our Website

About our website

The issue of endangered animals and species extinction has always been a major concern of animal conservationists and environmentalists alike. Unfortunately, throughout the years, there has been less and less care and concern for the protection and preservation of endangered animals. We chose this topic in order to promote and spread awareness of this issue. This task will be accomplished by making information about endangered animals and their threats more widely available

Tool used

Apiary: Documentation tool for our RESTful API, used to document our RESTful API. This is the link to our Apiary

Trello: Project management tool used to track tasks and issues. This is the link to our Trello

PlantItPoker: Project management tool used to estimate the amount of time a task would take to complete.

Namecheap: Domain name registration site used to obtain a name for our website.

Flask: Web microframework for backend Python that was used to connect our HTML templates to our URL address, as well as develop a RESTful API.

Github: Web - based Git version control software that was used to maintain the source code. This is our Github repo

Bootstrap: Front-end framework used to design our webpages.
